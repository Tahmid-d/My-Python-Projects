# My Python Expense Tracker

এই প্রজেক্টটি আমার পাইথন লার্নিং জার্নির একটি অংশ। এটি ব্যবহার করে প্রতিদিনের খরচ ট্র্যাক করা যায়।

## Features (কী কী সুবিধা আছে)
* প্রতিদিনের খরচের পরিমাণ ইনপুট নেওয়া যায়।
* খরচের ওপর ভিত্তি করে ব্যবহারকারীকে একটি রেটিং দেওয়া হয়।
* সব তথ্য `expense.txt` ফাইলে জমা থাকে।

## How to Run (কীভাবে চালাবে)
১. পাইথন ইনস্টল করা থাকলে কোডটি ডাউনলোড করো।
২. টার্মিনালে লেখো: `python my_code.py`

## My Goal
আমি ভবিষ্যতে একজন টেক লিডার বা CTO হতে চাই এবং এখন পাইথন দিয়ে লজিক বিল্ডিং শিখছি।
